apwlib
This is a collection of code that I've written in Python that I use in multiple projects. It's mostly relevant to Astronomy (in fact, I think it is currently just Astronomy-related), but the plan is to add other code as well. Feel free to report bugs, add suggestions, and email me if you'd like to contribute. Currently, there is no official documentation (although I expect that to change too), but some things you might find useful:

  - Date / coordinate conversion (MJD to Gregorian, RA/Dec to Galactic, etc.) (apwlib.convert, apwlib.geometry)
  - Plotting spectra by supplying a list of filenames, or a list of tuples in the format (wavelength grid, flux) (apwlib.spec.Plot)
  - Overlaying spectral lines on a matplotlib plot (apwlib.spec.Lines)
  - A wrapper around Python's datetime.datetime object that has mjd and jd attributes, and other astronomically useful times (apwlib.astrodatetime)
